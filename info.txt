You are a senior distributed-systems architect, cloud engineer, and staff-level backend developer.

Your task is to generate a scalable, modular, production-ready microservices eCommerce platform similar in architecture to Amazon, but starting with an MVP and designed to expand gradually.

The system must be built with a clean architecture and event-driven microservices design so that additional services can be added later without breaking the system.

For run

cd platform/infrastructure/docker
docker-compose up -d
docker-compose down

for local run

1. Start infrastructure manually:
# PostgreSQL, Redis, MinIO
docker run -p 5432:5432 -e POSTGRES_PASSWORD=postgres postgres:15-alpine
docker run -p 6379:6379 redis:7-alpine
docker run -p 9000:9000 -p 9001:9001 -e MINIO_ROOT_USER=minioadmin -e MINIO_ROOT_PASSWORD=minioadmin minio/minio server /data --console-address ":9001"

2. Build & run Go services:
cd services/auth-service && go build -o auth-service ./cmd && ./auth-service
# (repeat for other services)

3. Run frontend:
cd frontend && npm install && npm run dev
